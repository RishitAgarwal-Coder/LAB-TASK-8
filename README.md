# **Doubly and Circular Linked List Lab Tasks**

This repository contains C programs for managing dynamic data using Doubly Linked Lists (DLL) and Circular Linked Lists (CLL).

**1. Employee Record Management (DLL):**
- Features a doubly linked list where records are inserted at the front.
- Demonstrates bidirectional pointer management (`prev` and `next`).
- Prints the list state after every insertion to track changes in real-time.

**2. Dynamic DLL Memory Management:**
- Focuses on appending elements and efficiently removing the tail node.
- Ensures memory is freed correctly using `free()` to prevent leaks.

**3. Positional Insertion (CLL):**
- A circular linked list where nodes are inserted at a specific user-defined position $M$.
- Implements wrap-around logic to maintain the circular nature of the list.

**4. Supermarket Queue (CLL):**
- Models a real-world supermarket queue using a circular linked list.
- Implements a "remove last" feature to manage customer exits from the tail of the queue.



**Objective:**
Mastering complex pointer logic, including bidirectional traversal and circular link maintenance, to build more robust and flexible data structures.
